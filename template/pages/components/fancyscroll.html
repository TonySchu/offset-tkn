Fancyscroll

Spin-v1.0.0 includes a smooth scrolling plugin which allows you to animate the page's scrolling to elements within the page. The markup snippet below demonstrates:

Example
Scroll to the #example-section below 
<a href="#example-section" data-fancyscroll data-offset="70">Scroll to the #example-section below</a>
The above link will scroll to this section.
